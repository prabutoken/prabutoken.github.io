Why PRABU

PRABU is a decentralized derivatives primitive for risk-based applications built on the Binance blockchain. Through a series of native protocols, PRABU provides the framework and infrastructure for a robust, fully-functional decentralized exchange for derivatives contracts (futures and options) and parimutuel contracts.


Network value accrues from transaction fees generated within the protocols. One hundred percent of the transaction fees are shared between staked PRABU token holders, specialized node operators, active participants, network treasury, and developer pools vital to the network ecosystem and functionality.
